# Getting Started with toby-ui

Toby-Ui contains React components, React hooks, and SDK methods.

# Installation

1. `npm install @ybot1122/toby-ui`
2. Add toby-ui to `tailwind.config.js`:

```js
module.exports = {
  //...
  content: [
    // ...
    "node_modules/@ybot1122/toby-ui/dist/**/*.js",
  ],
};
```
